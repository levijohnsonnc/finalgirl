# Official Lore / Flavor Style Notes

Use this file only when short thematic writing is useful.
The primary job of this skill is accuracy, not creative writing.

## Tone Target

Aim for:
- pulp slasher VHS horror
- ominous
- cinematic
- lean
- atmospheric
- a little grimy
- playful only in a dark, genre-aware way

Avoid:
- parody unless requested
- fanfiction energy
- over-written prose
- lore invention presented as canon
- "movie trailer voice" excess in routine answers

## Usage Rules

Only use this tone in:
- short app microcopy
- optional section headers
- category labels
- brief flavor blurbs
- non-canon atmosphere around official facts

Never let theme distort:
- rules
- product contents
- release status
- component counts
- official lore claims

## Safe Uses

Good:
- "The night is about to get worse."
- "One killer. One location. One terrible plan."
- "Survive the film. If you can."

Bad:
- inventing official backstory
- stating motives or lore details that are not verified
- adding fake canon around killers, locations, or final girls

## Style Notes

Keep it:
- short
- tactile
- tense
- visual
- restrained

Preferred rhythm:
- short sentences
- vivid nouns
- minimal adjectives
- no purple prose

## Canon Rule

If a lore point is not supported by official material:
- do not state it as fact
- either omit it or label it as non-canon flavor
