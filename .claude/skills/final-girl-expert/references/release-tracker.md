# Release Tracker

This file exists to prevent the skill from treating uncertain products as released.

## Rule

If release status is not verified from an official source, do not present it as established fact.

## Tracking Format

### [Product Name]
- **Status:** released / announced / unknown
- **Verified from:** official source name or URL
- **Verified on:** YYYY-MM-DD
- **Notes:** optional

## Use in Answers

When asked about "all released content":
1. Prefer the exact tracked entries in this file and the product catalog
2. If this file is incomplete, say the high-level line is verified but the full itemized release list may still need refresh
3. Never silently fill gaps

When asked about a specific product not listed here:
- Say you cannot confirm its release status from a verified source
- Direct the user to vanrydergames.com to check directly
